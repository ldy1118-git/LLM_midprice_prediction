**보고일**: 2026-03-18

## 1. 실험 목적

기존 LightGBM 모델이 63거래일 후 종가를 예측하고 있으나, **시작가에서 예측 종가까지의 중간 경로는 제공하지 못함.** LLM을 활용하여 62일간의 일별 종가 경로를 생성함으로써, 사용자에게 "이 종목이 어떤 흐름으로 예측 가격에 도달하는지"를 시각적으로 보여주는 것이 목표.

## 2. 실험 설계

| 항목 | 내용 |
|------|------|
| 예측 구간 | 2024-04-01 ~ 2024-07-01 (62 거래일) |

| 조건 | 시작일/종료일 종가 고정, 중간 62일 예측 |

### 실험 변수 (총 75가지 조합)

- **과거 데이터 기간**: 1년 / 2년 / 3년 / 4년 / 5년
- **피처 세트**: A(close만) / B(OHLCV) / C(+기술지표) / D(+매크로) / E(전체 52컬럼)
- **LLM 모드**: thinking_expand / auto / instant

### 평가 지표

MAE, MAPE, RMSE, Pearson/Spearman 상관계수, DTW, 방향 정확도, 변동성 비율, 선형보간 대비, Brownian Bridge 대비

### 초기 관찰

1. **Brownian Bridge 대비 대부분 우수** (85%) — LLM이 단순 확률 모델보다 의미 있는 경로를 생성
2. **선형 보간 대비는 혼재** (55%) — close만 사용한 최소 피처(A)에서도 절반 이상 개선
3. **변동성 과소추정 심각** (Vol Ratio = 0.171) — LLM이 실제보다 훨씬 매끄러운 경로를 생성하는 경향
4. **종목별 편차 큼** — Pearson r이 -0.17(CIFR) ~ 0.96(CRNC)으로 넓은 범위
