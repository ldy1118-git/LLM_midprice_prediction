1. 배경
주가 예측은 패턴에 대한 이해, 금융 지식 등이 종합적으로 활용되어야하는 매우 고난이도의 지적 활동이다. 생성형 AI의 벤치마크 척도가 될 수 있다.

2. 실험
생성형AI가 주가 예측을 얼마나 정확히 할 수 있는지 예측한다.
D1~D63 (63일)의 캔들 값(시가 종가 고가 저가) 들을 입력으로 준다
D126일의 캔들 값(시가 종가 고가 저가)을 입력으로 준다.
D64~D125일의 캔들 값을 예측하여 출력하게 한다.
실험 데이터셋은 2016년~2025년 십년 중에서 나스닥의 종목 중 랜덤 샘플링해서 데이터 20개 데이터셋을 구성하고 5회씩 반복실험 진행.
AI가 특정 종목을 검색하여 출력하지 못하도록 종목과 날짜는 비식별화 마스킹.

생성형 AI는
chatGPT 5.4 (26년 4월 20일 기준) 채팅을 사용
Gemini 3.1 Pro (26년 4월 20일 기준) 채팅을 사용
Claude opus 4.7 (26년 4월 20일 기준) 채팅을 사용
를 비교함

[실험1]
숫자  메트릭스 데이터를 입력으로 주고, 숫자 메트릭스 데이터(csv)를 출력하도록 함
변동성,차트의패턴 고려를 해서 평균적인 변동성과 일반적인 캔틀패턴을 고려해서 값을 생성
좀 현실적이게 차트의 다이나믹한 움직임을 반영할 수 있게 함

비교 메트릭
1. 수치비교 (20개 데이터별로 5회 반복 실험 = 100회 실험에 아래 값을 계산해서 평균하여 표로 정리)
1) 종가 MSE = MSE (정답 데이터의 종가 62일 리스트 - 예측 데이터의 종가62일 리스트)
2) 표준화 종가 MSE = MSE (정답 데이터의 종가 62일 리스트 - 예측 데이터의 종가62일 리스트)를 (정답 데이터의 종가 62일리스트) 예측시작일종가(D64)와 마지막일종가(D125)를 선형계산했을 때의 종가 62일 리스트)의 MSE로 표준화
3) 일일변동성 MSE (정답 데이터의 고가-저가 변동성 62일 리스트 - 예측 데이터의 고가-저가 변동성 62일 리스트)
4) 표준화 일일변동성 MSE = 3에 대해서 2의 표준화를 적용하여 계산함

2.시각화 비교
각 예측 데이터를 캔들 차트로 시각화 출력하여 시각적으로 비교함

[실험2]
입력 데이터를 캔들차트 이미지로 만들고 중간 예측부분을 빨간 박스로 표지한 이미지를 입력으로 주고
중간 캔들을 예측하여 채워진 캔들 차트 이미지가 출력되게 함

비교 메트릭
1. 성공률 비교
(실험 결과 대부분 성공에 실패하였음)
1) 1단계 성공률 : 무한 로딩되거나 이미지 생성 안되고 텍스트 출력되면 실패로 간주 = (성공 회수 / 100회)
2) 2단계 성공률 : 이미지가 생성되었으나 캔들이 62개의 80%이상 출력되지 않으면 실패로 간주 = (성공 회수 / 100회)

2단계 성공률이 5%가 안되어서 다른 비교는 수행하지 않음
-> 실패 사진의 결과를 시각적으로 제시함

4. 해석
상대적으로 변동성을 매우 약하게 예측한다.
Claude > Gemini > ChatGPT의 예측성능이 가장 우수하다.
