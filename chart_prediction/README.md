# Chart-based Midprice Prediction

캔들차트 이미지를 LLM에게 제공하고, 3개월 뒤 종가를 예측하게 하는 실험.

## 목적

텍스트(수치) 기반 예측(`text_prediction/`)과 달리, **차트 이미지만으로** LLM이 주가 흐름을 읽고 미래 가격을 예측할 수 있는지 검증.
트레이딩뷰 스타일 캔들차트를 입력으로 사용하며, 차트의 빈 영역(뒤 3개월)에 해당하는 종가를 예측.

## 실험 설계

### 데이터
- **차트 구간**: 6개월 (앞 3개월 캔들 표시 + 뒤 3개월 비어있음)
- **종목**: 20개 (대장주 10 + 랜덤 10)
- **입력**: 앞 3개월 캔들차트 + 마지막 종가 박스 표시 (Y축 오른쪽)
- **정답**: 차트 끝(3개월 뒤) 빨간 점으로 표시된 실제 종가

### 종목 선정 기준

- **후보 풀**: `stock_panel_data.parquet` 내 LightGBM 예측 대상 약 1,000개 티커
- **필터 조건**: 해당 6개월 구간에 실제 종가 데이터가 빠짐없이 존재하는 종목
- **연도 분산**: 2020~2024년 각 연도에서 4개씩 선정 (대장주 2 + 랜덤 2)
  - 다양한 시장 환경(상승장, 하락장, 횡보장)을 포함하기 위함
- **대장주 10개**: NASDAQ 대형 기술주 중 대표 종목 직접 선정
- **랜덤 10개**: 위 조건을 만족하는 종목 중 무작위 샘플링

## 폴더 구조

```
chart_prediction/
├── scripts/
│   ├── generate_candle_chart.py      # 캔들차트 생성 (트레이딩뷰 스타일)
│   └── predict.py                    # API 예측 스크립트
├── data/
│   ├── input/                        # blank_3m 차트 (LLM 입력용)
│   │   └── {TICKER}_blank_3m.png
│   └── label/
│       ├── actual_midprices.csv      # 정답 종가 (ticker, last_visible_close, actual_end_close)
│       └── full_charts/              # full_6m 차트 (GT 확인용)
│           └── {TICKER}_full_6m.png
├── prompts/                          # 프롬프트 템플릿
├── output/                           # LLM 예측 결과
│   └── {model}/{variant}/
│       ├── predictions.csv           # 파싱된 예측 (ticker, predicted_price)
│       └── raw/                      # API 원본 응답 (JSON)
│           └── {TICKER}.json
├── evaluation/                       # 평가
│   ├── evaluate.py
│   └── results/
│       └── {model}/{variant}/
├── reference/                        # 트레이딩뷰 참고 스크린샷
└── README.md
```

## 사용법

### 1. 차트 생성
```bash
python scripts/generate_candle_chart.py
```

### 2. 예측 실행
```bash
python scripts/predict.py --model gemini --variant flash
```

### 3. 평가
```bash
python evaluation/evaluate.py output/gemini/flash/
```
